# Data-Quality-Checker
This is data quality checker that searches for null values, duplicates &amp; performs schema validation if user chooses so.
<br/>
<br/>
Upload a CSV file to analyze data quality. The system will identify how many null values exist, along with their corresponding columns and rows. It will also detect duplicate rows and indicate which entries are duplicates of an inital record.
